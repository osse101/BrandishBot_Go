# Patch Notes: February 27, 2026

Welcome to the latest update! We've been hard at work behind the scenes to improve the game experience, squash bugs, and lay the groundwork for exciting future features. Here's a summary of what's changed over the past week:

## Gameplay & Features

* **Job System Enhancements:** We've introduced a new system for handling job unlocks and bonuses! This means job progression will feel more rewarding and balanced, with dynamic bonus modifiers now in place. We've also updated the XP amounts for various activities.
* **Farming Updates:** Harvests now have tier limits, and we've refined the underlying logic to make farming more robust.
* **Composting Improvements:** We've added more comprehensive checks and balances to the compost system to ensure it works smoothly, preserving your compost bin's capacity correctly during deposits.
* **Platform Usernames:** Added support for platform-specific usernames (like Twitch or Discord) to user profiles, making it easier to link accounts and see who's who. Level-up events will now proudly display your username!
* **Inventory Tracking:** We've added better tracking for inventory events, which will help us monitor the economy and balance the game more effectively.

## Under the Hood (Developer & Admin Updates)

While you might not see these changes directly, they make the game run better and help us build new features faster!

* **Devtool Upgrades:** Our internal developer tools got a major boost. We've added a new `scenario` command for testing different game situations, integrated a log analysis tool, and added a watch mode for checking code coverage.
* **Better Testing:** We've spent a lot of time writing new tests for quests, economy transactions, crafting, harvesting, and the compost system. This means fewer bugs and a more stable game for everyone.
* **Code Organization:** We've done a lot of "spring cleaning" in our codebase! Packages like crafting, economy, expedition, and harvest have been reorganized to be cleaner and easier for our team to work with.
* **Documentation:** We've updated our internal documentation to match all these recent changes, making sure our team is always on the same page.
* **Validation & Security:** We've tightened up how the game handles requests, adding more validation to ensure everything runs securely and as expected.
