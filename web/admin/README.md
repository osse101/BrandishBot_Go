# Admin Dashboard

This directory contains the source code for the BrandishBot Admin Dashboard, a web-based interface for managing the bot.

## Documentation

For detailed usage instructions, configuration guide, and deployment scenarios, please refer to the main documentation:

**[Admin Dashboard Usage Guide](../../docs/features/ADMIN_DASHBOARD_USAGE.md)**

## Quick Start

To build the dashboard as part of the application:

```bash
# From project root
make admin-build
```

To run the development server for the frontend only:

```bash
cd web/admin
npm install
npm run dev
```
